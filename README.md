## 🎯 ¿Qué es AulaUTN?

**AulaUTN** es una plataforma web tipo *repositorio académico* desarrollada específicamente para profesores y alumnos de la Universidad Tecnológica de Nayarit. Permite **subir, organizar, compartir y gestionar archivos digitales** (PDF, DOCX, XLSX, PPTX, etc.) mediante publicaciones tipo *post*, sin necesidad de editar los archivos originales dentro de la plataforma, pero con control total sobre los metadatos del contenido publicado.

---

## ✨ Funcionalidades principales

### 📤 Subir archivos
- Sube archivos en múltiples formatos (PDF, DOCX, XLSX, PPTX, etc.)
- Organiza los archivos por materia, unidad o semana
- Vista previa rápida del contenido antes de publicar

### 📝 Publicaciones (Posts)
- Cada archivo se acompaña de un **post** con:
  - Título y descripción
  - Etiquetas o categorías
  - Fecha de publicación
  - Instrucciones adicionales para los alumnos
- Los posts se pueden **editar** (solo metadatos, no el archivo en sí)
- Los posts se pueden **eliminar** (junto con el archivo asociado)

### 📥 Descargar archivos
- Los alumnos pueden descargar los archivos directamente desde cada post
- Acceso rápido desde cualquier dispositivo
- Sin límites de descarga

### 🗑️ Eliminar contenido
- Elimina posts completos (incluyendo el archivo adjunto)
- Confirmación de seguridad antes de eliminar
- Solo profesores y administradores tienen permisos de eliminación

### ✏️ Editar posts
- Edita título, descripción, etiquetas y metadatos del post
- **No se edita el archivo original** (solo su información asociada)
- Historial de cambios visibles para los profesores

### 📁 Organización avanzada
- Carpetas virtuales por materia, trimestre o unidad
- Búsqueda por nombre, etiqueta o fecha
- Filtros por tipo de archivo

## 👥 Roles de usuario

| Rol | Permisos |
|---|---|
| **Profesor** | Subir archivos, crear/editar/eliminar posts, organizar contenido, gestionar carpetas |
| **Alumno** | Visualizar posts, descargar archivos, ver metadatos, buscar contenido |
| **Administrador** | Gestionar usuarios, materias, configuración general y supervisar todo el contenido |
